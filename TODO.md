# FinTree — Группировка задач по доменным областям

---

## Identity / Onboarding / Пользователь

**Контекст:** регистрация, Telegram login, первый вход, первичная инициализация данных.

### Проблемы

- Нет редиректа после использования Telegram-виджета
- После регистрации через Telegram нет обучения, отображается пустая Аналитика (до refresh)
- Некорректный формат email при регистрации через Telegram
- В обучении отсутствует настройка категорий
- Обучение не завершается корректно (не исчезает без обновления страницы)
- В сайдбаре не отображается имя пользователя при входе через Telegram

### Системные дефекты

- Отсутствует корректный post-auth flow
- Нет флага `IsOnboarded` / завершения обучения
- Данные не инициализируются после регистрации
- Нарушен lifecycle onboarding-сценария

## Accounts (Счета)

**Контекст:** управление счетами, балансами и их UI-представлением.

### UI / UX

- Скрывать фильтр если счетов меньше 5
- Убрать заголовки для упрощения визуала
- Полный редизайн модального окна добавления счета

### Доменная логика

- Пересмотреть концепцию начального остатка (заменить на корректировки?)
- Первый созданный счет автоматически делать основным
- Баланс и корректировки — неочевидный механизм

### Консистентность состояния

- Баланс не обновляется после создания транзакций
- Баланс не обновляется после добавления корректировки

---

## Transactions (Транзакции)

**Контекст:** создание, валидация и навигация по транзакциям.

- Валидировать дату транзакции относительно даты создания счета
- Сломаны переходы в транзакции при кликах на пиковых днях и расходах по категориям

---

## Analytics (Аналитика)

**Контекст:** визуализация метрик, корректность расчетов, UX отображения.

- Переименовать "Баланс" на странице Аналитики
- Не показывать графики и метрики при недостаточном количестве транзакций (минимум 5–10 дней данных)
- Динамику расходов строить с даты первой транзакции
- Изменение капитала начинает расчет с несуществующих дат

---

## Investments (Инвестиции)

**Контекст:** инвестиционные счета, доходность, отображение UI.

- Пока нет инвестиционных счетов — не показывать остальные элементы страницы
- Неправильно рассчитывается общая доходность
- Добавить архивирование инвестиционных счетов
