# Product Context

## Mission
Help people control personal finances without complex tooling: fast transaction capture, clear analytics, practical behavior change.

## Audience
Regular users, not analysts. Understandable metrics, simple flows.

## Scope boundaries
- No cross-user analytics (no plans to add).
- No mandatory bank integrations.
- Investment module: high-level only, no instrument-level accounting.

## Navigation (confirmed)
- Default authenticated page: /analytics.
- No dashboard module. No income plan.
- Category management in profile/settings.
- User popup: Settings + Logout only.

## Subscription
- Payment history: append-only, stored for all users.
- Full access with active subscription; read-only without.
